# UM CCE Intramurals 2026 — Login Web

A simple PHP login and sign-up page for the University of Mindanao CCE-Trojans Intramurals 2026.

## Files
- `login.php` — Login page
- `signup.php` — Registration page
- `CCE_2026.png` — Sports poster (right panel)
- `um_logo.png` — UM seal logo

## How to Run
1. Copy all files into `C:\xampp\htdocs\LabAct_loginWeb\`
2. Start Apache in XAMPP
3. Go to `http://localhost/LabAct_loginWeb/login.php`

## Requirements
- PHP 8.0+
- Apache (XAMPP / WAMP)

## Notes
- All image files must be in the same folder as the PHP files.
- Database connection is not yet implemented (TODO in both PHP files).
